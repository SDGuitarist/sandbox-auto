# Sandbox — Full Automation Experiments

This is a throwaway sandbox repo for testing full automation with Claude Code.

## Rules
- No secrets, no credentials, no real data in this repo
- No remote — do NOT push anywhere
- Experiment freely — nothing here matters
- Follow compound engineering phases even in auto mode (that's what we're testing)

## What We're Testing
- Can Claude Code run a full compound phase autonomously?
- Where does it go off the rails without permission prompts?
- How well does plan alignment hold without human checkpoints?
